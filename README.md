## Setup Steps

- Clone Repository

```
https://github.com/DileepGhanta/Quiz-App.git
```
- Go to directory

```
cd Quiz-App
```
- Install Dependency 
```
npm install
```
- Start LocalHost Server
```
npm run dev
```
